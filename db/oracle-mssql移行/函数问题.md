# ssma转换bug

ssma 会将

```sql
 -- set @aa = 1
 select CONCAT(@aa, '00') from dual;
 -- 结果 '100'
```

转换成

```sql
-- set @aa = 1
select ((ISNULL(@aa, '')) + '00');
-- 结果 '100'
```

## 原因

### oracle concat

```sql
	select CONCAT(1, '00') from dual;
	-- 结果 '100'
```

### sql server concat

```sql
select CONCAT(1, '00')
-- 结果 '100'
SELECT (1 + '00')
-- 结果 1 sql server的优先级转换规则，会首先将文字'00'转换为数字0
```
